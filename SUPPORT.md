# Support

## Getting Help

- **Questions**: Use [GitHub Discussions](https://github.com/DanOps-1/bingo-light/discussions)
- **Bug Reports**: Open an [Issue](https://github.com/DanOps-1/bingo-light/issues/new?template=bug_report.md)
- **Feature Requests**: Open an [Issue](https://github.com/DanOps-1/bingo-light/issues/new?template=feature_request.md)
- **Security**: See [SECURITY.md](SECURITY.md)

## Quick Diagnostics

If something isn't working:

```bash
bingo-light doctor          # Full diagnostic
bingo-light status          # Check fork health
bingo-light version         # Verify installation
```
