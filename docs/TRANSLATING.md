# Translation Guide

## Adding a new language

1. Copy `README.md` to `README.<locale>.md` (e.g., `README.ja.md`)
2. Add your language to the switcher line in ALL README files
3. Translate content — see guidelines below

## Guidelines

- **Natural voice**: Write as a native developer would, not a word-for-word translation
- **Keep verbatim**: Code blocks, JSON examples, badge URLs, command names
- **Adapt tone**: "Fork maintenance sucks" should feel equally punchy in your language
- **Sync marker**: Add `<!-- synced: <git-hash-of-english-readme> -->` at the top to track drift
